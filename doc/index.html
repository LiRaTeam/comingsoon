Set Up How to set up the theme?

Set up the theme

The theme has many settings. You can change them in "assets/js/config.js"

1) Setup the countdown

Find the "end_date" and change the value of it by rules below 

/* ------------------ The countdown settings --------------------------------- */
// Formate of date: MOUNTH DAY, YEAR HOURS:MINUTES:SECONDS
// time to count down to
var end_date = 'April, 30 2014 12:00:00';

// Target time in Los Angeles, USA (GMT -08:00): 
// If timezone is left at null it defaults to the timezone on the client.
var tm_zone = -4;
× Note! Don't forget to change the timezone to yours otherwise in other parts of the world the countdown will end before (or later) than you need.
2) How to set the page which opens when countdown ends..

You just need to change "expiry_goto" on the address of page which must be opens, when countdown ends. Or set "expiry_goto" equal "null" if you don't want open any page.

6) How to set up a subscription form?

The script saves emails to the file "emails.csv" .
